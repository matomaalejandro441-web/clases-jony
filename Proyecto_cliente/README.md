API CRUD con FastAPI

Descripción del proyecto

Este proyecto consiste en el desarrollo de una API REST utilizando FastAPI en Python.

La aplicación permite realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre:

Clientes
Facturas
Transacciones

Además, se implementaron modelos utilizando Pydantic para validar la información recibida y enviada por la API.


Tecnologías utilizadas

Python
FastAPI
Pydantic
Uvicorn
